### Prefixes (columns)

| Prefix    | Category      | English equivalent                              |
| --------- | ------------- | ----------------------------------------------- |
| **ki-**   | Interrogative | _wh-_ words: what, where, when…                 |
| **ti-**   | Demonstrative | _that / those_ (pointing to something specific) |
| **i-**    | Indefinite    | _some-_ words: something, somewhere…            |
| **ĉi-**   | Universal     | _every- / all_ words: everything, everywhere…   |
| **neni-** | Negative      | _no- / none_ words: nothing, nowhere…           |

> **Tip:** Add **ĉi** before or after a _ti-_ correlative to make it proximal ("this" instead of "that"):  
> `tio` = _that thing_ → `ĉi tio` = _this thing_  
> `tie` = _there_ → `ĉi tie` = _here_

### Suffixes (rows)

| Suffix  | Category           | Core idea                                  |
| ------- | ------------------ | ------------------------------------------ |
| **-o**  | Thing              | objects, concepts                          |
| **-u**  | Individual         | people or a specific one from a group      |
| **-a**  | Kind / Quality     | type or nature of something                |
| **-e**  | Place              | location                                   |
| **-en** | Motion (direction) | movement toward a place (accusative of -e) |
| **-el** | Way / Manner       | how something is done                      |

## Grammar Notes

### Plural & Accusative

Correlatives ending in **-o** and **-u** behave like nouns — they can take the plural **-j** and accusative **-n** endings:

| Base             | Plural            | Accusative | Plural Accusative |
| ---------------- | ----------------- | ---------- | ----------------- |
| tiu (_that one_) | tiuj (_those_)    | tiun       | tiujn             |
| kio (_what_)     | —                 | kion       | —                 |
| ĉiu (_everyone_) | ĉiuj (_all, pl._) | ĉiun       | ĉiujn             |

### "Any-" forms with _ajn_

There is no separate _"any-"_ prefix. Instead, add **ajn** after an indefinite (_i-_) or interrogative (_ki-_) correlative:

| Form      | Meaning               |
| --------- | --------------------- |
| `io ajn`  | _anything_            |
| `iu ajn`  | _anyone_              |
| `ie ajn`  | _anywhere_            |
| `kio ajn` | _whatever_            |
| `kiu ajn` | _whoever / whichever_ |
| `kie ajn` | _wherever_            |

### Amount correlatives as noun phrases

**-om** correlatives can head a noun phrase with **da**:

> `iom da akvo` = _some water_  
> `kiom da homoj?` = _how many people?_  
> `neniom da tempo` = _no time at all_
