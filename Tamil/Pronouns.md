### 2. Object Pronouns (The Receivers)

These are used when an action is being done _to_ the pronoun (e.g., "Tell _me_", "Call _him_").

| English Equivalent        | Tamil Script      | Transliteration       |
| :------------------------ | :---------------- | :-------------------- |
| **Me**                    | என்னை             | Ennai                 |
| **Us** (Inclusive)        | நம்மை             | Nammai                |
| **Us** (Exclusive)        | எங்களை            | Engalai               |
| **You** (Informal)        | உன்னை             | Unnai                 |
| **You** (Formal / Plural) | உங்களை            | Ungalai               |
| **Him** (Informal)        | அவனை / இவனை       | Avanai / Ivanai       |
| **Her** (Informal)        | அவளை / இவளை       | Avalai / Ivalai       |
| **Him / Her** (Formal)    | அவரை / இவரை       | Avarai / Ivarai       |
| **It**                    | அதை / இதை         | Adhai / Idhai         |
| **Them** (People)         | அவர்களை / இவர்களை | Avargalai / Ivargalai |
| **Them** (Things)         | அவற்றை / இவற்றை   | Avarrai / Ivarrai     |

### 3. Possessive Pronouns (Ownership)

These denote ownership (e.g., "My book", "Her pen"). _Note: Tamil often uses a short suffix form for possession, but these are the base possessive adjectives._

| English Equivalent         | Tamil Script           | Transliteration         |
| :------------------------- | :--------------------- | :---------------------- |
| **My**                     | என் / என்னுடைய         | En / Ennudaiya          |
| **Our** (Inclusive)        | நம் / நம்முடைய         | Nam / Nammudaiya        |
| **Our** (Exclusive)        | எங்கள் / எங்களுடைய     | Engal / Engaludaiya     |
| **Your** (Informal)        | உன் / உன்னுடைய         | Un / Unnudaiya          |
| **Your** (Formal / Plural) | உங்கள் / உங்களுடைய     | Ungal / Ungaludaiya     |
| **His** (Informal)         | அவன் / அவனுடைய         | Avan / Avanudaiya       |
| **Her** (Informal)         | அவள் / அவளுடைய         | Aval / Avaludaiya       |
| **His / Her** (Formal)     | அவர் / அவருடைய         | Avar / Avarudaiya       |
| **Its**                    | அதன் / அதனுடைய         | Adhan / Adhanudaiya     |
| **Their** (People)         | அவர்கள் / அவர்களுடைய   | Avargal / Avargaludaiya |
| **Their** (Things)         | அவற்றின் / அவற்றினுடைய | Avarrin / Avarrinudaiya |

### 4. Interrogative Pronouns (Question Words)

Notice how interrogative pronouns in Tamil generally start with **E** or **Ya**, matching the Proximate (**I**) and Distant (**A**) patterns.

| English Equivalent | Tamil Script | Transliteration | Notes                         |
| :----------------- | :----------- | :-------------- | :---------------------------- |
| **Who**            | யார்         | Yaar            | Used for people               |
| **What**           | என்ன         | Enna            | Used for things or actions    |
| **Which (one)**    | எது          | Edhu            | Singular (for things/animals) |
| **Which (ones)**   | எவை          | Evai            | Plural (for things/animals)   |
| **Whose**          | யாருடைய      | Yaarudaiya      | Possessive question           |
